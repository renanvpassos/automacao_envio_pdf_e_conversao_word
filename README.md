#Sistema de Envio de arquivos pdf por e-mail com opção de converter para word

Tecnologia: Python

- Selecionar mês (altera corpo do e-mail e assunto
- Selecionar se quer converter os pdfs para word antes de enviar
- Anexar documentos nos botões de acordo com o nome do funcionario
- Se selecionado converter pdf para word surge campo com botão para converter os pdfs e visualizar conversões
- Botão para disparar os emails
